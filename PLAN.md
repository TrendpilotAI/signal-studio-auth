## Architecture Overview

signal-studio-auth is a drop-in authentication layer for the Signal Studio FastAPI backend.
It provides:
- **Dual-mode JWT middleware** — accepts Supabase or legacy ForwardLane tokens
- **Auth API routes** — proxies to Supabase Auth API (signup, login, logout, refresh, invite)
- **User mapping** — converts Supabase UUID claims to the legacy int-ID User model
- **RLS migrations** — Postgres row-level security for multi-tenant org isolation
